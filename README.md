## Minimal Path to Awesome

- Clone this repository
- Ensure that you are at the solution folder
- in the command-line run:
  - **npm install**
  - **gulp serve**

## Features

- Using the "Details List" fluent UI component to display data from the list. 
- Single/Batch update for items through buttons
